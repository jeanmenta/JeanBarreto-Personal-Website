Create a personal web page in GitHub

  • Sign up for a GitHub account if you don't already have one.
  • Create a new repository on GitHub by clicking on the "+" button in the top right corner of the homepage and selecting "New repository". Give the repository a name (e.g. "my-website") and make sure it is set to public.
  • In the repository, go to the "Settings" tab. Scroll down to the "GitHub Pages" section and select the branch you would like to use for GitHub Pages (usually "master").
  • Now that your repository is set up for GitHub Pages, you can start adding files to it. The easiest way to do this is to create a new file called "index.html" in the root of your repository. This file will be the starting point for your website.
  • Write the HTML, CSS and JavaScript code for your webpage and save it into the index.html file. If you're unsure of how to do this, you can find many tutorials online that will walk you through the basics of web development.
  • Commit the changes to the repository by clicking on the "Commit Changes" button at the bottom of the page.
  • Your website should now be live at a URL like https://username.github.io/my-website (replace "username" with your GitHub username and "my-website" with the name of your repository).
  • You can also customize the domain name of your website (e.g. www.example.com) and link it to your GitHub page.
  • Finally, you can continue to make updates to your website by editing the files in your repository and committing the changes.
