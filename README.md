# PersonalWebsite

## Description
A simple personal website with a guide on how to setup a personal GitHub website. 

## Installation
To run the website, you will need to have a basic understanding of HTML and CSS. You can clone or download the repository and open the `index.html` file in a web browser to view the website. 

## Usage
You can use this website as a starting point to create your own personal website. You can customize the HTML and CSS files to suit your own needs and style.
